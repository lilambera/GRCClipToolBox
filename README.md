# 管人吃自用烤肉工具集

仅自用，上传仅作存档用
因为自用所以做得很简陋
有很多冗余和未处理的报错情况可以优化，但短时间内不会优化抱歉

------------------------------------------------------------------------

## 介绍

一个用于 **视频处理和字幕烧录** 的桌面工具。
使用开源代码时要注意将ffmpeg和deno依赖包下载并放在主文件夹内

------------------------------------------------------------------------

## 功能

-   字幕烤录到视频功能
-   油管视频下载功能（可批量扒直播窗）
-   基于whisper的字幕生成功能（需自己下载模型）
  
-   一些小功能，包括：
-   ASS->TXT
-   油管字幕重断句（只支持txt格式输入，毕竟重新打轴太不现实，不如直接whisper）
-   时间戳提取（用于先轴后翻，轴完了让whisper生成参考文本用的）
   （但这边内置的whisper字幕生成功能不支持这样搞）
-   油管cookies获取
