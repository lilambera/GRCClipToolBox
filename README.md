## 介绍

一个用于 **视频处理和字幕烧录** 的桌面工具。\

------------------------------------------------------------------------

## 功能

-   字幕烤录到视频功能
-   油管视频下载功能（可批量扒直播窗）
-   基于whisper的字幕生成功能（需自己下载模型）
-   一些小功能，包括：
-   ASS->TXT
-   油管字幕重断句（看过的都清楚那破字幕有时候断句成啥稀碎样子了）
-   时间戳提取（用于先轴后翻，轴完了让whisper生成参考文本用的）
   （但这边内置的whisper字幕生成功能不支持这样搞）
-   油管cookies获取
